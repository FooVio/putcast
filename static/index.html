generate RSS feeds from Put.io




Put.io has taken Putcasts functionality inside. Here's how you can find the RSS feed of any folder:




One caveat: Putcast was removing the need to authenticate. So all your feeds were public. put.io couldn't allow that. So the RSS feeds will require authentication, (if requested from another IP). Programs like iTunes ask for username and password, but If you're going to use a NAS like a Synology for example, be sure to add your username and password into the RSS url like this: https://username:password@put.io/rss/all/11231231
